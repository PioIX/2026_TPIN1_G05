## Descripción de la propuesta
Este juego se llama WorldQuiz. Es un juego en el cual se busca saber el conocimiento de las personas sobre las banderas del mundo. El objetivo será que los jugadores identifiquen correctamente, a partir de la bandera, el nombre del mismo. La funcionalidad principal es mostrar la imagen y ofrecer varias respuestas cuando el usuario lo vea necesario (presionando un botón el cual dirá: “Ayuda”) pero su costo será de cinco puntos. El jugador deberá escribir o seleccionar el país correcto dentro de un tiempo determinado. A medida que avance, los niveles van a subir de dificultad mostrando países menos conocidos. De esta manera, el juego resultará entretenido y desafiante para personas de diferentes edades.  El sistema de puntaje sumará 10 puntos por cada respuesta correcta, pero si el jugador utiliza el botón de “Ayuda”, se le sumarán únicamente 5 puntos. Si responde incorrectamente, no se sumarán ni se restarán puntos.


## Alcance
- Juego individual (Single-player).
- Sistema de preguntas sobre banderas de países del mundo.
- Base de datos con países, puntajes y respuestas correctas.
- Sistema de puntaje: +10 puntos por respuesta correcta.
- Botón de ayuda que muestra opciones de respuesta y reduce la recompensa a +5 puntos.
- Aumento de niveles de dificultad.
- Registro de jugadores y puntajes.
- Pantalla de inicio, juego y resultados finales.
- Ranking de mejores puntajes.
